 WorksWhale
</h1>

> 업무용 연락처임을 나타내면서 시원한 바다(파랑)의 느낌을 주기 위해 '고래'를 이름에 추가했다.
> 
> 앱의 목적은 업무용 연락처를 부서별로 저장하고 전화나 메세지 기능을 사용할 수 있게 이동하는 것이다.
>
> 개발 기간 : 24/01/15 ~ 24/01/19

<br/>

## 👨‍💻 프로젝트 팀원
한병철, 최성진, 오성원, 박희수
